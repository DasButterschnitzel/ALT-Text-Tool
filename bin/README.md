# bin/

Lege hier dein ExifTool-Binary ab. Das Tool zieht es bevorzugt aus diesem Ordner.

- Windows: `exiftool.exe` (download von https://exiftool.org)
- Linux/macOS: `exiftool` (z. B. `sudo apt install libimage-exiftool-perl`, dann `which exiftool` und Symlink hierher)

Falls hier nichts liegt, sucht das Tool im System-PATH.
